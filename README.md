# ♟️ Guess the Opening — Chess Quiz

An interactive chess opening quiz game built for [ChessLessonsPro.com](https://www.chesslessonspro.com/guess-the-opening/).

🎮 **[Play the game →](https://www.chesslessonspro.com/guess-the-opening/)**

***

## About

A browser-based quiz game where players identify chess openings and variations from real board positions. Built as a single HTML file with no dependencies except chess.js (CDN).

The game covers **70+ popular openings and named variations** — from beginner-friendly lines to sharp expert-level systems.

***

## Features

- ♟️ Accurate board rendering via [chess.js](https://github.com/jhlywa/chess.js)
- 🎯 70+ openings and variations with ECO codes
- 3 game modes: **Mixed**, **Families only**, **Variations only**
- 4 difficulty levels: **Beginner**, **Intermediate**, **Expert**, **All**
- 💡 Hint system (3 hints per round, affects score)
- 🌙 Dark theme optimized for chess aesthetics
- 📱 Fully responsive — works on desktop and mobile
- ⚡ Single HTML file — no build tools, no server required

## Scoring

| Result | Points |
|--------|--------|
| Correct, no hints | 10 pts |
| Correct, 1 hint used | 7 pts |
| Correct, 2 hints used | 4 pts |
| Wrong answer | 0 pts |

Maximum score per game: **100 points**

## Usage

### Standalone
Just open `chess-opening-quiz-v3.html` in any modern browser — no server needed.


### WordPress
Add a **Custom HTML** block with the iframe code above. See the [live implementation](https://www.chesslessonspro.com/guess-the-opening/) on ChessLessonsPro.com.
